## ⚡ Quick Actions

**On Trial Org Detail Page:**
- ➕ **Add Activity** - Log calls, demos, emails
- 📝 **Add Note** - Quick observations
- 🔄 **Update Stage** - Move trial forward
- 📅 **Set Timeline** - Trial start/end dates
- 🎫 **Create Ticket** - When things break

**Trial Stages (In Order):**
```
1. Lead → 2. Onboarding → 3. Demo → 4. Evaluation →
5. Negotiation → 6. Converted → 7. Churned
```

## 🔍 Search Tips

- Search by **org name**
- Filter by **stage**
- Sort by **trial start date**

## 🆘 Common Issues & Fixes

| Problem | Solution |
|---------|----------|
| Can't see my orgs | Refresh page, check you're not filtering |
| Forgot password | Profile → Change Password |
| Page looks weird | Hard refresh (Ctrl+F5 / Cmd+Shift+R) |
| Missing features | You might be "viewer" not "admin" |
| Something broke | Tell Sriharsha (gently) |

## 💡 Pro Tips

✅ **DO:**
- Add notes frequently
- Update stages as trials progress

## 🎯 What Each Role Can Do

**Account Managers (viewer role):**
- ✅ View assigned trial orgs
- ✅ Add notes and activities
- ✅ Update trial stages
- ✅ Create tickets
- ✅ View reports (filtered to their orgs)
- ❌ Cannot assign orgs
- ❌ Cannot manage users
- ❌ Cannot access admin features

**Admins:**
- ✅ Everything account managers can do
- ✅ View ALL trial orgs
- ✅ Assign orgs to account managers
- ✅ Manage users and permissions
- ✅ Access roadmap and admin tools
