   - Organization details (name, domain, stage, engagement score)
   - Trial start/end dates
   - Assigned sales POC (if any)

2. **The system automatically filters by:**
   - User role (Account Manager vs Admin)
   - Account Manager assignment (account_manager_id column)
   - Active trial status

3. **No additional configuration needed!**
   - Role-based filtering is built into the platform
   - Works on localhost and production
   - Applies to all relevant pages (trials, dashboard, reports)

## 📞 Support & Troubleshooting

### **Common Issues:**

**1. "Link has expired"**
- Signup links are valid for 7 days only
- Contact admin for a new signup link

**2. "Invalid token"**
- The link may have been used already
- Contact admin for a new signup link

**3. "Can't see my trial organizations"**
- Check if trial orgs have been imported yet
- Verify your email matches the Account Manager assignment in trial orgs
- Contact admin to verify account_manager_id assignment

**4. "Forgot my password"**
- Use the "Forgot Password?" link on the login page
- Password reset link will be sent to your email
